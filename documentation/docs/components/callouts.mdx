Specialized alert components for documentation with stronger visual emphasis.

<Callout variant="warning">This is a warning callout highlighting potential pitfalls.</Callout>

### Real-World Examples

<Callout variant="tip" title="Gas Optimization">
  Use <code>Temporary</code> storage for ephemeral data to reduce gas costs in your Soroban
  contracts.
</Callout>

<Callout variant="error" title="Security Warning">

</Callout>

## Usage

```tsx
import { Callout } from '@site/src/components/Callout';

<Callout variant="tip">
  Your helpful tip here
</Callout>

<Callout variant="warning" title="Custom Title">
  Important warning message
</Callout>
```
